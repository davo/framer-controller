Class HistoryController<T>

A Controller manages a component's props. It maintains a state that can be easily changed and which it passes to a controlled component as props. It also keeps a history of states and can take commands, such as undo and redo, to traverse this history. You can easily extend the Controller class to add custom controls for any component.
